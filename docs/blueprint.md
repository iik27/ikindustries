# **App Name**: IK Industries Profile

## Core Features:

- Profile Showcase: Display Muhamad Taufik Hidayat's profile information.
- Services Offered: Detail the services offered by IK Industries, focusing on technology, application development, website development, and AI solutions.
- Key Strengths: Highlight IK Industries' key strengths and unique selling points.
- Portfolio Display: Showcase Muhamad Taufik Hidayat's personal portfolio with relevant projects. Fetch and render high-quality images from Unsplash or other image sites.

## Style Guidelines:

- Primary color: Light ice blue (#B0E2FF) to represent technology and innovation, creating a calm yet modern aesthetic.
- Background color: Very light desaturated blue (#F0F8FF) to ensure the content is legible and clean. Its hue is similar to the primary color, but subdued.
- Accent color: A deeper shade of blue (#4682B4) to add depth and visual interest.  It should be far enough from the primary color that CTAs stand out.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines and short amounts of text and 'Inter' (sans-serif) for body text, to maintain a tech-forward, modern feel. Both fonts contribute to a contemporary and readable design.
- Responsive layout similar to Apple's themes, ensuring the website is accessible and visually appealing on all devices.
- Use simple, clean icons that complement the ice blue color scheme and enhance the user experience.
- Subtle animations for transitions and interactions, adding a modern touch to the website's UI/UX.
